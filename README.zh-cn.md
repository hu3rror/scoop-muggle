## 🛠️ 快速上手
1. 添加仓库：

在PowerShell里输入：

```pwsh
scoop bucket add muggle 'https://github.com/hu3rror/scoop-muggle.git'
```

2. 安装软件：

```pwsh
scoop install muggle/软件名
```

比如装个懒人版mpv：

```pwsh
scoop install muggle/mpv-lazy
```

3. 更新姿势：

```pwsh
scoop update *
```
